ご利用の手順（動画による説明がご覧いただけます）


ご利用の手順（動画が再生されない、詳しい説明はこちらをご覧ください）

ログイン

パソコン、携帯電話、またはスマートフォンからログイン用ページにアクセスします。IDとパスワードを入力してログインします。

新規予約・予約の確認・変更・キャンセル

新規予約画面でクルマ、予約開始時間、予約終了時間を登録します。登録したアドレスに予約受付メールが届きます。
予約の確認、変更、キャンセルも可能です。

利用開始の手順

  1. 予約時間の１４分前からご利用いただけます。
  2. 乗車前に、車体の傷やタイヤのパンク等が無いかご確認ください。
  3. モバイルサイト、またはカードタッチで利用開始操作を行いドアを開錠します。
    クルマのハザードランプが２回点滅し、ドアロックが解除されます。
  4. グローブボックス内にあるキーボックスからエンジンキーを取り出します。
  5. エンジンキーを用いて通常のクルマと同じように運転します。

利用中の注意事項

  • 利用開始から利用終了までの間は、一般のクルマと同じです。
  • クルマから一時的に離れる場合は、エンジンキーを使ってドアをロックし、戻ってきたらエンジンキーでドアを解除します。
  • 返却が遅れそうなときは予約時間内にモバイルサイトで予約の延長手続きを行ってください。次の予約が入っていない場合は２回まで延長ができます。

利用終了の手順

  1. 車両を所定の位置に駐車して、エンジンを停止します。
  2. エンジンキーをグローブボックス内のキーボックスコネクタに奥まで確実に差し込みます。
  3. 車内外のライトが消えていること、サイドブレーキを引いたことを確認します。
  4. 車内に忘れ物がない事を確認して降車し、ドアを閉めます。
  5. モバイルサイト、またはカードタッチで利用終了操作を行いドアを施錠します。
    クルマのハザードランプが１回点滅し、ドアが施錠されます。

困ったときは…

返却が遅れそうな場合…
モバイルサイトにアクセスして予約の延長が可能です。（利用開始後最大2回まで。次の予約がない場合に限る。）
延長できない場合は、車内掲載の電話番号へご連絡ください。
予約終了時刻の10分前に車内ブザーが鳴ります。その後も10分毎にブザーが鳴ります。
利用終了ができない場合…
ハザードランプが点滅し、車内ブザーが鳴ります。以下の点をご確認ください。
  • サイドブレーキは引いていますか？
  • エンジンは停止し、キーを抜いていますか？
  • キーはグローブボックスに戻っていますか？
  • ドアが半ドアになっていませんか？
確認後、利用終了手順をやり直してください。
忘れ物をした場合…
利用終了後10分以内であれば、モバイルサイトからドアロックの解除が1度だけ行えます。
モバイルサイトにアクセスして［忘れ物ドア開錠］ボタンを押してください。
